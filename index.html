jquery-scrollspy

A ScrollSpy library for jQuery that detects the enter and exit of elements in a viewport or scrollable div.

This documentation covers the latest version of ScrollSpy. The 0.2.x version tree has some minor breaking changes from the 0.1.x version tree. For stability, consider scrollspy.jquery-0.1.3.

Usage

To track elements that enter and exit the viewport:

$('.tile').on('scrollSpy:enter', function() {
  console.log('enter:', $(this).attr('id'));
});

$('.tile').on('scrollSpy:exit', function() {
  console.log('exit:', $(this).attr('id'));
});

$.scrollSpy('.tile');

To track elements that enter and exit the viewable region of a div with overflow:scroll:

$('#parent-element').scrollSpy('.tile');
